🔢 Aula 06 - Tipos de Dados: Números

🎯 Objetivo da Aula

Aprender a trabalhar com números inteiros e decimais em Python, realizar operações matemáticas, receber dados do usuário e converter tipos de dados.

📚 Conteúdos estudados

- int()
- float()
- input()
- Operações matemáticas
- Comparações numéricas
- Conversão de tipos
- import math
- math.sqrt()
- math.factorial()
- if e else
- Precedência dos operadores matemáticos.

📂 Arquivos desta aula

- README.md
- exemplo_numeros.py
- exercicio01.py
- exercicio02.py

🏋️ Exercícios

Exercício 01

Programa que realiza a soma de dois números digitados pelo usuário.

Exercício 02

Programa que calcula o valor final de um produto após a aplicação de um desconto percentual.

🚀 O que aprendi?

Ao concluir esta aula fui capaz de:

- Trabalhar com números inteiros e decimais.
- Utilizar input() para receber informações do usuário.
- Converter Strings em números utilizando int() e float().
- Realizar operações matemáticas.
- Utilizar funções do módulo math.
- Entender a utilização de condicionais em programas simples.
- Criar programas mais interativos utilizando dados digitados pelo usuário.

📝 Resumo

Esta aula ampliou os conhecimentos adquiridos nas aulas anteriores ao introduzir o trabalho com números em Python. Agora já é possível desenvolver programas capazes de receber informações do usuário, realizar cálculos e apresentar resultados de maneira dinâmica.

# 🧠 Aula 03 - Variáveis

## 🎯 Objetivo

Aprender o conceito de variáveis e como utilizá-las para armazenar informações em Python.

## 📚 Conteúdos estudados

- O que é uma variável.
- Como criar variáveis.
- Atribuição de valores com `=`.
- Regras para nomear variáveis.
- O Python identifica automaticamente o tipo da variável.
- Utilização de variáveis com a função `print()`.

## 💡 Conceitos importantes

Uma variável é um espaço na memória utilizado para armazenar um valor que poderá ser utilizado ao longo do programa.

Exemplo:

```python
nome = "Felipe"
idade = 28

print(nome)
print(idade)
```

## ✅ Aprendizados

Ao concluir esta aula fui capaz de:

- Criar variáveis.
- Armazenar textos e números.
- Alterar o valor de uma variável.
- Exibir o conteúdo de uma variável utilizando `print()`.
- Escrever nomes de variáveis de forma mais organizada.

## 🚀 Resumo

As variáveis são a base da programação. Elas permitem guardar informações para reutilizá-las durante a execução do programa, tornando o código mais organizado, reutilizável e fácil de manter.

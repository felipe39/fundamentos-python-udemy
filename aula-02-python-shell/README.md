# 🐍 Aula 02 - Python Shell

## 🎯 Objetivo

Aprender os primeiros comandos em Python utilizando o Python Shell e criar os primeiros programas.

## 📚 Conteúdos estudados

- Abrir o Python Shell
- Fazer operações matemáticas
- Utilizar a função `print()`
- Consultar ajuda com `help()`
- Criar arquivos `.py`
- Executar programas
- Entender o conceito de Case Sensitive

## 📂 Arquivos

- `primeiro_programa.py` — Primeiro programa em Python.
- `exercicio01.py` — Exibe o nome.
- `exercicio02.py` — Exibe a idade.
- `exercicio03.py` — Exibe o resultado de uma soma.
- `exercicio04.py` — Exibe uma mensagem na tela.
- `exercicio05.py` — Exibe um texto junto com o resultado de um cálculo.

## ✅ Aprendizados

Ao finalizar esta aula fui capaz de:

- Criar programas em Python.
- Utilizar a função `print()`.
- Executar arquivos `.py`.
- Fazer cálculos com Python.
- Organizar meus estudos utilizando Git e GitHub.
